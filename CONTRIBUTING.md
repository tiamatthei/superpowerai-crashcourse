# Contributing to SuperpowerAI Crashcourse

## How to contribute

1. Fork the repo
2. Create your feature branch (`git checkout -b feature/new-lesson`)
3. Commit your changes (`git commit -m 'Add lesson 9: Advanced MCP'`)
4. Push to branch (`git push origin feature/new-lesson`)
5. Open Pull Request

## Development setup

```bash
pip install -r requirements-dev.txt
pre-commit install
pytest
```

## Code style

- Black formatter
- isort
- mypy type checking
